# Glossário

| Termo | Definição |
|---|---|
| **Protocolo / Experimento** | Unidade de pesquisa cadastrada (ex.: PC1699). |
| **Fator** | Variável controlada do experimento (1 a 3). Ex.: produto, dose. |
| **Nível** | Valor de um fator. A combinação de níveis gera os tratamentos. |
| **Tratamento (T1..Tn)** | Combinação de níveis aplicada às parcelas. |
| **Parcela** | Unidade experimental; célula do croqui (ex.: 4,05 m × 7 m). |
| **Bloco / Repetição** | Agrupamento para controle de variação (DBC). |
| **Delineamento** | Esquema de casualização: DIC (inteiramente casualizado), DBC (blocos casualizados), Fatorial, Split-plot. |
| **Croqui** | Mapa das parcelas no campo (grid bloco × tratamento). |
| **Timing** | Momento/condição de uma aplicação ou avaliação (calendarizado ou condicional). |
| **Avaliação** | Variável-resposta coletada (ex.: produtividade, nº de capulhos). |
| **Unidade de coleta / saída** | Unidade em que se mede vs unidade reportada (ex.: kg/parcela → kg/ha). |
| **Área útil colhida** | Área efetivamente colhida da parcela = `nº linhas colhidas × espaçamento (cadastro) × comprimento colhido`; informada na colheita; base do cálculo de produtividade (RN-PROD). |
| **Ensaio** | Tipo de fluxo: interno (TCC/sem custo) ou comercial (com orçamento). |
| **CAD** | Instância de aprovação do protocolo (fluxo comercial). |
| **SAGRE** | Sistema (Shiny/R) de análise estatística de experimentos; fonte da estatística a portar. |
| **Conjunta** | Análise multi-local (vários experimentos/locais combinados). |
| **Letras (comparação de médias)** | Agrupamento de médias (Tukey, Scott-Knott…) por letras a/b/c. |
